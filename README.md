# macula-boot

该模块提供微服务应用开发的基础依赖库

```
01. macula-boot-starter-mapstruct                对象转换
02. macula-boot-starter-cache                    多级缓存
03. macula-boot-starter-redis                    REDIS（基于Redisson）
04. macula-boot-starter-lock4j                   分布式锁
05. macula-boot-starter-idempotent               幂等
06. macula-boot-starter-leaderelection           基于Redis的Leader选择器
07. macula-boot-starter-mybatis-plus             MyBatis Plus
08. macula-boot-starter-sender                   基于本地数据库发送MQ消息（先落库再异步发MQ）
09. macula-boot-starter-rocketmq                 RocketMQ
10. macula-boot-starter-liteflow                 可编排的组件引擎
11. macula-boot-starter-statemachine             基于Spring StateMachine的状态机
12. macula-boot-starter-security                 基于Token的安全校验
13. macula-boot-starter-skywalking               Skywalking集成
14. macula-boot-starter-springdoc                Swagger Doc v3
15. macula-boot-starter-web                      WEB
16. macula-boot-starter-tinyid                   ID生产
17. macula-boot-starter-system                   接入菜单和权限等系统服务
18. macula-boot-starter-powerjob                 基于PowerJob的定时任务（类似阿里云的ScheduleX）
19. macula-boot-starter-oss                      资源中心(minio/aliyun/cos/aws s3/本地)
21. macula-boot-starter-cloud                    Spring Cloud集成
    1. macula-boot-starter-feign                 Feign定制，使用OkHTTP
    2. macula-boot-starter-gateway               网关定制，基于Oauth2认证，权限控制也集中在网关
    3. macula-boot-starter-alibaba               基于alibaba spring cloud体系（nacos、sentinel、seata）
    4. macula-boot-starter-alibaba-scg           基于slibaba的gateway定制
    5. macula-boot-starter-tencent               基于tencent的polarismesh服务治理体系
    6. macula-boot-starter-tecent-scg            基于tencent的gateway定制
    7. macula-boot-starter-tsf                   基于TSF的服务治理体系
    8. macula-boot-starter-tsf-scg               基于TSF的网关定制
```
